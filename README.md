# Todo List

タスク管理用アプリケーション
![todo_list](https://user-images.githubusercontent.com/112733879/196066303-42871e7e-3d43-40b8-930d-b725e5ac94b8.png)

## 作成した目的

サーバ上で不特定多数のユーザがタスクを管理できるようにするためのアプリです

## 機能一覧

-   タスクの追加
-   タスクの更新
-   タスクの削除

## 使用技術（実行環境）

-   PHP 　 8.1.6
-   Laravel 　 8.83.25
